API Endpoints

POST /api/parse

Parse natural language description into structured parameters.

curl -X POST http://localhost:8000/api/parse \
  -d "description=a cube 50mm side"

POST /api/generate

Generate a 3D model from parsed parameters.

curl -X POST http://localhost:8000/api/generate \
  -d "description=a cylinder 80x100mm" \
  -d "shape=cylinder" \
  -d "diameter=80" \
  -d "height=100"

GET /api/examples

Get example natural language descriptions.

curl http://localhost:8000/api/examples

GET /download/{filename}

Download a generated model file.
